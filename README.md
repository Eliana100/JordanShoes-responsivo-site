Este projeto é uma Landing Page de uma loja de sneakers (JordanShoes), desenvolvida para consolidar conhecimentos em layouts complexos e responsividade avançada utilizando **CSS Grid**.

*Projeto no Figma:* https://www.figma.com/design/Yb9IBH56g7T1hdIyZ3BMNO/Desafios---CodeLab?node-id=1883-2&t=hwYKcRzgvjs5lavl-1

<br>

## 🌸 Desafios

<br>

Neste projeto, foquei em aplicar conceitos modernos de desenvolvimento Front-end:

- **CSS Grid Layout:** Implementação de uma malha flexível que altera a quantidade de colunas (1, 2 ou 3) automaticamente com base nos breakpoints de Media Queries.
- **Variáveis CSS (Custom Properties):** Centralização da paleta de cores e estilos no `:root`, facilitando a manutenção e garantindo consistência visual.
- **HTML Semântico:** Uso correto de tags como `<header>`, `<main>`, `<section>` e `<span>` para melhorar a acessibilidade e o SEO.
- **Responsividade Adaptativa:** Estruturação de breakpoints para Mobile, Tablet e Desktop (1200px+), garantindo que as imagens dos produtos não percam qualidade.

## 🍒 Visualização por Dispositivo

- **Mobile:** Layout em coluna única para facilitar o scroll.
- **Tablet:** Grid de 2 colunas para melhor aproveitamento do espaço.
- **Desktop:** Layout completo com 3 colunas e tipografia expandida.
